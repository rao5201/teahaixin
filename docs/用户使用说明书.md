## 1. 产品简介

**茶海心遇（TeaHaiXin）** 是一款 AI 驱动的情绪可视化社交应用。它将你的心情文字转化为唯美的意境图片、诗意文本和情绪背景音乐，帮助你更好地理解和表达自己的情绪。

### 核心特色

| 特色 | 说明 |
|------|------|
| AI 情绪分析 | 智能识别六种情绪：喜悦、忧伤、愤怒、平和、焦虑、浪漫 |
| 意境图片生成 | 根据情绪自动生成对应意境的艺术图片 |
| 诗意文本 | 匹配经典诗词与原创茶诗，为情绪赋予文学之美 |
| 情绪 BGM | 根据不同情绪播放对应氛围的背景音乐 |
| 情绪地图 | 在地图上查看周围人的情绪分布 |
| 情绪共振 | 发现与你拥有相同情绪的人，建立心灵连接 |
| 情绪周报 | 每周生成情绪分析报告，洞察内心变化 |

### 支持平台

- PWA 网页版（支持所有现代浏览器）
- Android 原生应用（APK / React Native）
- iOS 应用（React Native，通过 Expo）

## 2. 安装指南

### 2.1 PWA 网页版安装方法

PWA（Progressive Web App）网页版无需下载安装包，通过浏览器即可添加到主屏幕，使用体验接近原生应用。

#### Android 设备（Chrome 浏览器）

1. 打开 Chrome 浏览器，访问茶海心遇 PWA 地址
2. 页面加载完成后，浏览器底部可能自动弹出"添加到主屏幕"提示，点击 **添加** 即可
3. 若未自动弹出，点击浏览器右上角 **三个点菜单**
4. 选择 **"添加到主屏幕"** 或 **"安装应用"**
5. 在弹出的确认框中点击 **"添加"**
6. 应用图标将出现在手机主屏幕上，点击即可直接打开

#### iOS 设备（Safari 浏览器）

1. 打开 **Safari** 浏览器（必须使用 Safari，其他浏览器不支持 PWA 安装）
2. 访问茶海心遇 PWA 地址
3. 点击底部工具栏中间的 **分享按钮**（方框带向上箭头的图标）
4. 在弹出的分享菜单中向下滑动，找到并点击 **"添加到主屏幕"**
5. 在确认页面可以修改应用名称，然后点击右上角 **"添加"**
6. 应用图标将出现在主屏幕上

#### Windows 电脑（Chrome / Edge 浏览器）

1. 打开 Chrome 或 Edge 浏览器，访问茶海心遇 PWA 地址
2. 地址栏右侧出现安装图标（下载箭头或加号图标），点击该图标
3. 在弹出的确认框中点击 **"安装"**
4. 应用将以独立窗口打开，并可在桌面和开始菜单中找到快捷方式

#### Mac 电脑（Chrome / Edge 浏览器）

1. 打开 Chrome 或 Edge 浏览器，访问茶海心遇 PWA 地址
2. 点击地址栏右侧的安装图标
3. 在弹出的确认框中点击 **"安装"**
4. 应用将添加到 Launchpad 和 Applications 文件夹中

### 2.2 Android APK 安装方法

1. 从官网或指定渠道下载茶海心遇 APK 安装包
2. 打开下载的 APK 文件
3. 若系统提示"不允许安装未知来源的应用"：
   - 点击 **"设置"**
   - 开启 **"允许此来源安装应用"**
   - 返回安装页面
4. 点击 **"安装"**，等待安装完成
5. 点击 **"打开"** 即可使用

> **注意：** APK 安装包基于 React Native + Expo 构建，建议 Android 系统版本 6.0 及以上。

## 3. 功能使用指南

### 3.1 注册与登录

茶海心遇支持两种使用方式：

#### 注册账号

1. 打开应用，进入欢迎页面
2. 输入你的 **用户名**（2-20个字符）
3. 设置 **密码**（不少于6位）
4. 点击 **"注册"** 按钮
5. 注册成功后系统自动登录，并随机分配一个茶文化主题头像（如茶杯、樱花、明月等）

#### 登录账号

1. 在登录页面输入已注册的用户名和密码
2. 点击 **"登录"** 按钮
3. 登录成功后进入主页

#### 游客模式（PWA 网页版）

1. 在欢迎页面点击 **"随便看看"** 按钮
2. 系统将以"茶友"身份进入应用
3. 游客模式下可以体验情绪生成功能，数据保存在浏览器本地存储中
4. 游客模式不支持情绪地图、共振、周报等需要账号的功能

> **提示：** 登录状态将保持 7 天有效期，到期后需重新登录。

### 3.2 记录情绪

1. 进入主页（"心语"标签页）
2. 在文本输入框中输入你此刻的心情感受（最多 1000 字）
3. 也可以点击下方的 **快速心情标签** 快速填入预设文字
4. 点击 **"生成心语"** 按钮
5. 等待 AI 分析并生成结果

**支持的情绪类型：**

| 情绪 | 中文标签 | 触发关键词示例 |
|------|----------|----------------|
| happy | 喜悦 | 开心、快乐、幸福、棒、太好了、感恩 |
| sad | 忧伤 | 难过、伤心、孤独、思念、心碎、遗憾 |
| angry | 愤怒 | 生气、烦、讨厌、受不了、崩溃、不爽 |
| peaceful | 平和 | 平静、安宁、放松、舒适、自在、禅 |
| anxious | 焦虑 | 紧张、担心、害怕、压力、失眠、迷茫 |
| romantic | 浪漫 | 爱、心动、甜蜜、温柔、牵手、拥抱 |

> **提示：** 如果输入的文字未能匹配到特定情绪关键词，系统将默认识别为"平和"情绪。

### 3.3 查看 AI 生成结果

情绪分析完成后，结果页面将展示以下内容：

1. **情绪标签** - 显示识别出的情绪类型及对应的表情符号
2. **意境图片** - 由 AI 根据情绪生成的艺术风格图片（中国水墨风、自然风光等）
3. **诗意文本** - 与情绪匹配的经典诗词或原创茶诗
4. **情绪动画** - 页面背景呈现对应情绪的粒子动画效果：
   - 喜悦：金色星光粒子飘散
   - 忧伤：蓝色雨滴缓缓落下
   - 愤怒：红色火焰跳动
   - 平和：绿色茶叶悠然飘落
   - 焦虑：紫色漩涡缓缓旋转
   - 浪漫：粉色心形缓缓飘起

5. 在结果页面你可以：
   - 点击 **"收藏心语"** 将本次结果保存到历史记录
   - 点击 **"分享"** 将结果分享到社交平台
   - 点击 **"再来一次"** 返回主页重新输入

### 3.4 播放/暂停背景音乐

每次情绪生成结果都包含一首与情绪匹配的背景音乐（BGM）：

| 情绪 | 音乐风格 |
|------|----------|
| 喜悦 | 欢快旋律 |
| 忧伤 | 淡淡忧伤 |
| 愤怒 | 激昂乐章 |
| 平和 | 自然宁静 |
| 焦虑 | 舒缓解压 |
| 浪漫 | 浪漫时光 |

**操作方式：**

1. 在结果页面找到音乐播放区域
2. 点击 **播放按钮** 开始播放背景音乐
3. 再次点击可 **暂停播放**
4. 离开结果页面时，音乐会自动停止

### 3.5 分享到社交平台

1. 在结果页面点击 **"分享"** 按钮
2. 如果设备支持 Web Share API（大部分移动设备支持）：
   - 系统将弹出原生分享菜单
   - 可选择分享到微信、QQ、微博等已安装的应用
3. 如果设备不支持 Web Share API：
   - 系统将自动复制分享内容到剪贴板
   - 你可以手动粘贴到任意社交平台

**分享内容格式：**
```
[情绪表情] [情绪标签]

[诗意文本]

-- 茶海心遇 · 以茶会心，遇见情绪
```

### 3.6 历史记录查看与管理

#### 查看历史记录

1. 点击底部导航栏的 **"历程"** 标签（PWA 版）或 **"我的"** 页面（原生版）
2. 历史记录按时间倒序排列，显示每条记录的：
   - 情绪标签和表情
   - 生成的意境图片缩略图
   - 诗意文本摘要
   - 记录日期和时间
3. 点击任意记录可重新查看完整的生成结果

#### 管理历史记录

- **按情绪筛选**（原生版）：在历史记录页面选择特定情绪类型进行过滤
- **清除全部记录**：点击 **"清除记录"** 按钮，确认后删除所有历史数据
- 历史记录支持分页加载，每页默认显示 20 条

> **注意：** PWA 网页版的历史记录存储在浏览器本地，清除浏览器数据会丢失记录。注册账号后的记录保存在服务器端，不受浏览器影响。

### 3.7 个人主页与情绪统计

点击底部导航栏的 **"我的"** 标签进入个人主页，可以查看：

1. **个人信息**
   - 用户名和头像
   - 加入日期

2. **数据统计**
   - 情绪记录总数
   - 最常见的情绪类型
   - 连续记录天数（打卡天数）
   - 不同情绪类型数量
   - 共振次数

3. **情绪分布图表**
   - 以柱状图展示六种情绪的记录占比
   - 每种情绪的具体数量和百分比

4. **退出登录**
   - 点击 **"退出"** 按钮可注销当前登录状态

### 3.8 情绪地图功能

情绪地图允许你在地图上查看附近区域的情绪分布。

#### 查看附近情绪

1. 点击底部导航栏的 **"心图"** 标签
2. 应用将请求定位权限（需允许）
3. 地图上将显示附近用户发布的公开情绪记录
4. 每个情绪标记显示对应的表情符号和距离

#### 情绪热力图

1. 在地图页面切换到 **"热力图"** 模式
2. 可查看不同区域的情绪密度分布
3. 支持按情绪类型筛选
4. 支持选择时间范围（1-90 天）

#### 上报位置

- 在生成情绪结果时，可选择附带地理位置信息
- 位置信息用于在情绪地图上展示
- 可以选择是否公开显示

> **提示：** 附近情绪的默认搜索半径为 5 公里，最大可扩展到 50 公里。

### 3.9 共振功能

共振功能帮助你发现与你拥有相同情绪的用户，建立心灵连接。

#### 寻找共振

1. 系统会自动匹配 7 天内发布过相同情绪的其他用户
2. 显示匹配用户的用户名、头像、情绪内容和诗意文本
3. 每次最多匹配 20 位用户

#### 创建共振

1. 在共振列表中找到感兴趣的用户
2. 点击 **"共振"** 按钮
3. 系统将建立你们之间的共振连接
4. 对方将收到实时通知："[你的用户名] 与你产生了情绪共鸣"

#### 查看我的共振

1. 在共振页面可查看所有共振记录
2. 支持按类型筛选：
   - **全部** - 查看所有共振记录
   - **发出的** - 你主动发起的共振
   - **收到的** - 别人向你发起的共振
3. 共振记录支持分页浏览

> **注意：** 不能与自己产生共振，同一条情绪记录也不能重复共振。

### 3.10 情绪周报

每周系统会为你生成一份情绪分析报告。

#### 查看周报

1. 进入个人主页或周报页面
2. 周报内容包括：

| 模块 | 说明 |
|------|------|
| 统计周期 | 最近 7 天的起止日期 |
| 记录总数 | 本周情绪记录总数 |
| 环比变化 | 与上周记录数的对比（上升/下降/持平） |
| 主导情绪 | 本周最频繁的情绪类型 |
| 情绪丰富度 | 本周涉及的不同情绪种类数量 |
| 情绪分布 | 各情绪类型的数量和百分比 |
| 每日分布 | 按日期展示每天的情绪记录 |
| 共振统计 | 本周发出和收到的共振次数 |
| 情绪洞察 | 基于主导情绪生成的个性化建议 |
| 茶语金句 | 每期附赠一句与茶相关的人生感悟 |

## 4. 常见问题 FAQ

### Q1: 意境图片加载很慢怎么办？

**A:** 意境图片由 AI 实时生成（通过 Pollinations.ai 服务），生成速度取决于网络状况和服务器负载。建议：
- 确保网络连接稳定
- 耐心等待几秒钟，图片正在生成中
- 如果长时间未加载，可返回重新生成
- 页面会显示占位图，表示图片正在加载

### Q2: 背景音乐无法播放？

**A:** 可能的原因和解决方法：
- **浏览器自动播放限制：** 大部分浏览器要求用户先与页面交互后才能播放音频，请手动点击播放按钮
- **网络问题：** 音乐文件托管在 Pixabay CDN 上，请确保网络可以访问
- **静音模式：** 检查手机是否处于静音或振动模式
- **浏览器兼容性：** 建议使用 Chrome、Safari、Edge 等主流浏览器

### Q3: 数据是如何保存的？

**A:**
- **PWA 网页版（未登录）：** 数据保存在浏览器的 localStorage 中，清除浏览器数据会导致数据丢失
- **注册用户：** 数据保存在服务器数据库中，更换设备后登录同一账号即可恢复数据
- 建议注册账号以确保数据安全

### Q4: 情绪识别不准确怎么办？

**A:** 情绪识别基于关键词匹配和 AI 模型分析。如果识别结果不够准确，建议：
- 使用更明确的情绪描述词语
- 增加情绪相关的关键词（参见 3.2 节的触发关键词示例）
- 如果文字中未包含明确的情绪关键词，系统会默认识别为"平和"

### Q5: 忘记密码怎么办？

**A:** 目前系统尚未提供自助找回密码功能。如需重置密码，请联系管理员协助处理，或重新注册一个新账号。

### Q6: 可以删除某一条历史记录吗？

**A:**
- **PWA 网页版：** 目前仅支持清除全部历史记录
- **原生版：** 历史记录保存在服务器端，目前不支持单条删除，可联系管理员处理

### Q7: 生成失败提示"请稍后重试"？

**A:** 可能的原因：
- 服务器暂时不可用，请等待几分钟后重试
- 输入内容超过 1000 字限制，请缩短文字
- 网络连接异常，请检查网络设置

### Q8: PWA 安装后无法离线使用？

**A:** PWA 网页版通过 Service Worker 提供基本的离线缓存。首次访问时需要联网加载资源，之后部分页面和功能可以离线使用。但情绪生成功能需要联网访问 AI 服务。

## 5. 隐私与安全说明

### 数据安全

| 项目 | 说明 |
|------|------|
| 密码存储 | 使用 bcrypt 加密算法存储，即使数据泄露也无法还原原始密码 |
| 身份认证 | 使用 JWT（JSON Web Token）进行身份验证，令牌有效期 7 天 |
| 数据传输 | 建议部署 HTTPS 以确保传输加密 |
| 情绪数据 | 默认设置为公开可见，你可以选择设置为私密 |

### 隐私政策

1. **信息收集：** 我们仅收集注册时提供的用户名和密码，以及你主动输入的心情文字和可选的位置信息
2. **位置信息：** 情绪地图功能需要地理位置权限，该信息仅用于展示附近情绪和热力图，你可以随时拒绝授权
3. **数据公开：** 情绪记录的公开/私密状态由你自行控制，私密记录不会出现在公共列表和地图上
4. **数据删除：** 你有权要求删除个人数据，可联系管理员处理
5. **第三方服务：** 应用使用以下第三方服务：
   - Pollinations.ai - 图片生成
   - Pixabay - 背景音乐资源
   - HuggingFace（可选）- AI 情绪分析模型

### 安全建议

- 请设置不少于 6 位的强密码，避免使用简单数字组合
- 不要在心情文字中输入身份证号、银行卡号等敏感个人信息
- 在公共设备上使用完毕后，请及时退出登录
- 定期检查并清理不需要的历史记录
